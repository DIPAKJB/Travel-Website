# 🌍 Travel Website (TravelQuest)

## 📌 Project Description
TravelQuest is a modern travel website developed using the MERN stack. It allows users to explore destinations, view travel packages, and book trips easily through a user-friendly interface.

## 🚀 Features
- Explore travel destinations
- Booking system
- Responsive design
- API-based data (mockAPI)
- Contact page

## 🛠 Tech Stack
- Frontend: React.js (Vite)
- Backend: mockAPI
- Tools: VS Code, Git, GitHub

## 📂 Project Structure
- frontend → React application
- docs → report & screenshots
